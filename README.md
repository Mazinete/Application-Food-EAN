# App Todo Cours BAD 22.1

Petit exemple de mini application React

# Installation

Lancer la commande : 

```
git clone https://github.com/plarrat/App-todo-cours.git
```

Se positionner en ligne de commande dans le répertoire

Lancer la commande 

```
npm install
```

Démarrer le serveur

```
npm start
```
